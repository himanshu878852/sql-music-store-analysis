# SQL Music Store Analysis

## Project Overview

This project analyzes a digital music store database using SQL to answer real-world business questions related to sales performance, customer behavior, revenue generation, and music trends.

The goal is to demonstrate SQL proficiency through data exploration, business analysis, and insight generation.

---

## Business Problem

Music stores generate large amounts of transactional data across customers, invoices, tracks, artists, and genres. The challenge is to transform this raw data into actionable business insights that can support decision-making.

This project answers key business questions such as:

* Which customers generate the highest revenue?
* Which countries contribute the most sales?
* What are the best-selling genres?
* Which artists generate the highest revenue?
* What are the top-performing tracks?
